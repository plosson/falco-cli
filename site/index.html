falco

CLI for your Falco account — sync received peppol invoices, issued sales invoices and payment status from the terminal.

Install

curl -LsSf https://raw.githubusercontent.com/plosson/falco-cli/main/site/install | sh

macOS (Apple Silicon) and Linux (x64 / arm64). Single self-contained binary, no runtime required.

Commands

falco loginAuthenticate against your Falco account
falco whoamiShow the authenticated user
falco peppol listList received peppol documents
falco peppol get <id>Fetch one document (UBL XML, optional embedded PDF)
falco peppol syncIdempotent bulk sync of peppol documents into a folder
falco peppol mark-paid <id>Set an invoice's payment status
falco invoices syncSync issued sales invoices into a folder
falco updateSelf-update to the latest release

Update

falco update
